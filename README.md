## 📌 Project Overview

This is a console-based Bank Management System developed using Python. The application simulates basic banking operations, allowing users to create accounts, log in, deposit money, withdraw money, check account balances, and view account details.

Although this project is simple compared to my recent work, it represents an important milestone in my learning journey and the beginning of my programming experience.

---

## 🎯 Project Objective

The goal of this project was to practice core Python programming concepts by building a simple banking application from scratch.

During this project, I learned:

- Object-Oriented Programming (OOP)
- Classes & Objects
- Functions & Methods
- Conditional Statements
- Loops
- Dictionaries
- Input Validation
- Menu-driven Programs
